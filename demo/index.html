wasm-audio-decoders

Browser and NodeJS Web Assembly audio decoder libraries that are highly optimized for size and performance.

HTML Demo Source for this demo

This is the demo page for the wasm-audio-decoders library. Checkout the README of this repository on GitHub for more information.

How to use the demo

  1. Select a couple mp3 or ogg opus files to decode.
  2. If you want to download a WAVE file for each file that you decode, check the Download Results box
  3. Select the number of workers you wish to use.
  4. Click decode.
